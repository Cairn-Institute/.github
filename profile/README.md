# CAIRN Institute

CAIRN Institute develops educational, workforce, technology, research, and community-impact solutions that expand opportunity through responsible innovation.

## What we build here

This organization hosts the source code, technical documentation, and automation behind CAIRN Institute's public websites, student portals, Moodle integrations, AI agents, data-engineering projects, and workforce-development laboratories.

## How this organization is run

- Source code, version control, and releases live here on GitHub.
- - Non-technical project coordination happens in Linear/Basecamp.
  - - Architecture summaries, policies, and SOPs live in Notion.
    - - Official contracts, personnel, and financial records live in Google Drive/SharePoint, never in GitHub.
